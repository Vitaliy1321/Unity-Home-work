# Unity-Home-work
Я решил попробовать создать колыбельную Ньютона на Unity. Решил попробовать сделать это через "нити" с помощью Configurable Joint. Также я использовал Sphere Collider, чтобы задать материал и упругость шарам. С помощью Rigidbody, которую я использовал на шарах и нитях. Для шаров, чтобы задать им общую массу и задать общее направление для них. Для нитей это нужно было, чтобы задать кинематику для них. Также я добавил скрипт, который задает удар первого шара на постоянной основе с определенной силой и кнопку "R", которая позволяет замедлять скорость действия в 5 раз. Это получилось не совсем правильно и не совсем так, как я хотел, но я решил оставить эту попытку, так как она получилась лучше всего.
